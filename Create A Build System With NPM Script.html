Create A Build System With NPM Scripts

STEP 1. Create project and Enter into it

  1. Open Bash Terminal
  2. Make Project Directory
  3. CD into that directory
$	~/myFolder>	mkdir 	<dirName>
$	~/myFolder>	   cd 	<dirName>

STEP 2. Initialize and Run NPM

  1. Initialize the node package manager
  2. Change to parent directory
  3. Create vscode workspace
$	~/myFolder/project>	 npm init
$	~/myFolder/project>	  cd ../ 		
$	~/myFolder>			code <dirName>

STEP 3. Confirm package.json exists

Package.json File Sturcture Example

//package.json
    {
     //...
     "devDependencies":  {
     "building-static-server":  "*"
     },
     "scripts":  {
     "build":  "make build",
     "start":  "building-static-server -p 3001"
     }
     //...
    }

STEP 4. Add css Templating Folders

  1. Create scss subdirectory
  2. Create build subdirectory
$	~/myFolder/project>	mkdir scss
$	~/myFolder/project>	   cd build

STEP 5. Create file index.html + boilerplate code

  1. Add index.html to the build subdirectory
  2. Insert boilerplate html from jsbin.com
$	~/myFolder/project>	touch build/index.html

jsbin.com/?html,js,output Boilerplate code example
